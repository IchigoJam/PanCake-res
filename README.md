# PanCake-res

IchigoJam用マルチメディアボード、[PanCake](http://pancake.shizentai.jp/)のリソースをまとめます

## data

- [PanCake パレット](PANCAKE_COLORS.js) - 16色パレット
- [PanCake スプライト](sprites.csv) - 4bpp 8x8 HEX表現

## how to convert

```sh
deno -A download.js
deno -A png2txt_all.js
```
